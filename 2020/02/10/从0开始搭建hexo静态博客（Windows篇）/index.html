从0开始搭建hexo静态博客（Windows篇）

本文最后更新于：2021年1月1日 晚上

前言

身为可爱的蓝孩子（也可以是铝孩子啦）的你，是不是也想拥有一个和哥哥一样酷的博客呢？今天，我就来教你啦

准备条件

  1. 一个樱云账号，用于加快对github的访问速度
  2. 一个你自己的域名(也可以没有)
  3. 一双手
  4. 一个脑子
  5. 一台能上网的电脑
  6. 一个github账号

搭建过程

点击这里下载git

点击这里下载nodejs

下载完这俩玩意之后安装
创建一个文件夹，此文件夹非常重要，请妥善保存
然后在这个文件夹里面右击你的鼠标，选择“Git Bash Here”

然后你会打开一个新世界
在弹出窗口中输入

npm install -g hexo-cli

稍等一段时间即安装成功hexo(如果失败请使用三玖云加速)
安装成功后输入

hexo init

初始化博客安装

然后，你的这个文件夹里面会多出来这一堆东西，请务必保存好，这就是你的博客数据

(删库跑路警告）

然后访问github注册一个账号

已有账号的直接登陆即可，然后新建一个储存库

按如下填写用户名后创建

复制你的仓库git地址，打开你电脑上当时创建的博客文件夹，找到_config.yml文件，打开编辑器

按照图片内容结合个人修改信息

在最后的deploy一栏中，改为以下内容

deploy:
type: git
repo： 你刚才复制的git仓库地址
branch: master

保存退出

依旧是在博客文件夹右击选择git bash here

输入

npm install hexo-deployer-git –save
hexo g
hexo d

然后在弹出窗口输入你的GitHub用户名和密码，当出现

INFO Deploy done: git

时，即创建成功

创建后找到github仓库最后一栏设置

下滑找到Github Page,

按图操作

然后你可以通过

你的用户名.github.io

访问你的博客

使用教程

依旧时博客目录右击选择 git bash here

依次输入

hexo new 文章名称

然后再你的博客文件夹的source_posts文件夹可以找到该文件，使用Markdown语法编辑即可

然后执行

hexo clean
hexo g
hexo d

将其重新渲染并部署到github

进阶使用

你可以在github仓库页面找到设置，下滑找到github page绑定自己的域名

在你的域名供应商处设置解析

你还可以访问这里下载使用主题，不同主题使用方法各有差异，可以参考主题发布者的介绍

注意事项

遇到不知名报错或者不懂得地方请百度解决
